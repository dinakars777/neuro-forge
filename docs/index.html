Powerful Features

🧠

Smart Git Analysis

Analyzes your active Git tree to find exactly what you're working on right now.

📊

Real Token Counting

Shows exact token count using GPT-4 tokenizer. Warns when you exceed context limits.

🕐

Time-Based Selection

Grab files changed since "2 days ago" or "last Monday" with --since flag.

🎯

Target Specific Files

Bypass Git and pack exactly what you want with --files flag.

💬

Custom Prompts

Append your question with --prompt to paste everything in one shot.

📁

File Output

Write to a file instead of clipboard with --out flag.
